Fitur Utama

Semua yang anda butuhkan
untuk membuat laporan profesional

Dari desain visual hingga export multi-format — JDRP Report menangani semuanya.

Visual Designer

Drag & drop elemen langsung ke canvas. Resize, snap ke grid, multi-select, dan undo/redo penuh.

snap multi-select undo/redo grid

Data Binding Dinamis

Koneksikan field langsung ke data source. Gunakan expression engine untuk kalkulasi dan format otomatis.

{field} expression grouping

CrossTab Report

Buat laporan pivot otomatis dengan group header, footer, dan subtotal per kelompok data.

pivot subtotal group

Chart Interaktif

Integrasi Chart.js dengan 6 tipe chart — bar, line, pie, doughnut, radar, polar. Data dinamis dari source.

bar line pie +3

Peta Interaktif

Tampilkan data geografis dengan jsVectorMap. Region click untuk URL atau hint tooltip. Export sebagai gambar.

World Indonesia USA +10

Barcode & QR Code

30+ tipe barcode via bwip-js — CODE128, QR, EAN, UPC, PDF417, DataMatrix, dan banyak lagi.

CODE128 QR EAN-13 +27

Import JasperReports

Import file .jrxml JasperReports langsung ke JDRP. Konversi otomatis elemen, band, dan expression ke format native.

.jrxml band expression auto-convert

Cara Kerja

Dari desain ke laporan
dalam 4 langkah

01
Rancang layout
Buka designer, drag elemen ke canvas, atur band header-detail-footer sesuai kebutuhan.
→
02
Bind data
Hubungkan field ke data source JSON. Tulis expression untuk kalkulasi dan format dinamis.
→
03
Preview live
Viewer merender laporan secara real-time. Navigasi antar halaman, zoom, dan filter data.
→
04
Export & cetak
Pilih format PDF, Excel, Word, atau cetak langsung — semua elemen visual terkonversi sempurna.
Teknologi

Dibangun di atas library terpercaya

Semua dependensi diload lokal dengan CDN fallback — bekerja bahkan tanpa koneksi internet.

Rendering

bwip-js

Barcode generator — 30+ symbology, render ke Canvas, export PNG.

Visualisasi

Chart.js

Chart interaktif — bar, line, pie, radar, doughnut, polar area.

Peta

jsVectorMap

SVG map engine — region click, tooltip, warna custom per region.

Export Excel

ExcelJS

Workbook native — embed image, cell style, formula, freeze row.

Export Word

docx.js

DOCX native — TableCell, ImageRun, TextRun, paragraph styling.

Export PDF

html2pdf.js

HTML-to-PDF via Canvas capture — multi-page, custom margin.
